# Page: Task Queue Progress
**Route:** `/queue`

## Layout & Functionality
- **Header:** "System Operations Queue".
- **Visual List:** A vertical timeline or list of tasks generated by the active job search run.
- **State Colors (Strict Requirement):**
  - **Completed Tasks:** Background/Icon must be Green (`text-green-600` or `bg-green-100`).
  - **Ongoing Task:** Background/Icon must be Orange (`text-orange-500`), ideally with a subtle CSS pulse animation.
  - **Incoming/Pending Tasks:** Background/Icon must be Slate Gray (`text-slate-400`).
- **Data Displayed:** Action type (Scraping, AI Parsing), Target (Company Name), and timestamp.
